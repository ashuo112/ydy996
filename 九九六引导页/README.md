## 简介

九九六辅助网引导页，采用霓虹艺术风格设计，提供官网链接、商城链接和交流群等导航功能。

## 技术栈

- HTML5
- CSS3 (SCSS)
- JavaScript (jQuery)
- Canvas 动画效果

## 功能特性

- ✨ 霓虹艺术风格设计
- 🔗 分类导航链接
- 📱 响应式设计，适配各种设备
- 🎨 动态发光效果
- 🌸 樱花飘落动画
